Support

Help for the most common PackSnap questions.

This page exists as the public support destination for PackSnap. It is intentionally separate from the in-app experience.

Back to Overview Privacy Policy

Restore purchases

If PackSnap Pro does not appear unlocked, open the app and use Restore Purchases from the subscription or paywall flow. Purchases are managed by Apple and must be restored using the same Apple Account that completed the original purchase.

Widget questions

The PackSnap Home Screen widget is a Pro feature. If the widget looks empty or locked, open the app first so it can refresh its shared trip snapshot, then confirm that PackSnap Pro is active on the device.

Trip history and templates

Free users can view trip history. Pro unlocks advanced actions such as editing or reusing completed trips, traveler-based packing, the Home Screen widget, and additional template and item limits.

Data and deletion

PackSnap stores packing data on-device. To remove content, delete trips inside the app or remove the app from the device.

Public support route

For App Store listing purposes, this page is the PackSnap support URL. If you publish this site in a dedicated GitHub repository, enable GitHub Issues on that repository and link to it here as the public contact path.
